Comprendre JavaScript : Les bases essentielles pour débuter

1. Introduction à JavaScript

JavaScript est un langage de programmation qui permet de rendre les pages web interactives. Créé en 1995, il est devenu incontournable pour le développement front-end, mais aussi back-end grâce à Node.js.

Contrairement à ce qu'on pense souvent, JavaScript n'a aucun rapport avec le langage Java. Son nom a été choisi pour des raisons marketing. Aujourd'hui, c'est l'un des langages les plus populaires au monde.

« JavaScript est le langage du web. Si vous voulez créer des applications interactives, vous DEVEZ le maîtriser. » — Kyle Simpson

2. Les variables (var, let, const)

En JavaScript, on déclare des variables avec let, const ou var.

// Avec let (mutable)
let age = 25;
age = 26; // OK

// Avec const (immuable)
const nom = "Jean";
nom = "Pierre"; // ❌ Erreur

// var (à éviter)
var ancienne = "déprécié";

Règle d'or : Utilisez const par défaut, et let quand la variable doit changer. Fuyez var.

3. Les fonctions

JavaScript propose plusieurs façons d'écrire des fonctions.

// Déclaration classique
function direBonjour(nom) {
    return `Bonjour ${nom}`;
}

// Expression de fonction
const direAuRevoir = function(nom) {
    return `Au revoir ${nom}`;
};

// Fonction fléchée (moderne)
const direSalut = (nom) => `Salut ${nom}`;

Les fonctions fléchées sont plus courtes et ne redéfinissent pas le mot-clé this, ce qui évite certains bugs.

4. Conditions et boucles

Conditions

const age = 18;

if (age >= 18) {
    console.log("Majeur");
} else {
    console.log("Mineur");
}

Boucles

// for classique
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// forEach sur tableau
const fruits = ["pomme", "banane", "orange"];
fruits.forEach(fruit => console.log(fruit));

💡 Astuce : Privilégiez map(), filter() et reduce() pour manipuler des tableaux plutôt que for.

5. Conclusion

JavaScript est un langage vaste mais extrêmement gratifiant. En maîtrisant les bases (variables, fonctions, conditions, boucles), vous pourrez déjà créer des interactions simples sur vos pages web.

Prochaines étapes : manipuler le DOM, les événements, les promesses et async/await.

N’hésitez pas à pratiquer en faisant des petits projets : une calculatrice, un générateur de citations, ou une to-do list.
